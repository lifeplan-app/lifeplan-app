🖼 OGP画像エクスポートツール

📋 手順

  1. 各画像をブラウザで開く（右クリック → 新しいタブで開く）
  2. ブラウザの「印刷」→「PDFとして保存」 または スクリーンショットで1200×630pxで保存
  3. PNG形式で ogp/lifeplan-ogp.png / ogp/spending-ogp.png として保存
  4. index.html の og:image の URL を実際のドメインに書き換える

⚠️ Facebook・X（Twitter）・LINE は PNG/JPEG のみ対応。SVG は非対応です。
推奨ツール: Figma（SVGをインポートしてPNGエクスポート）または Squoosh・CloudConvert
