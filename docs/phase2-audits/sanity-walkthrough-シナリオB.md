## 検出された違和感（Phase 2 他監査との突き合わせ）

### 02-I01（現役期インフレ未適用）
- **観察**: 2026年 annualExpense 432 → 2055年も annualExpense 432（30年間同額）。2026年の月36万円生活費が 30年後も名目同額のまま。
- **判定**: ✅ I01 のバグが現実の年次でそのまま観察される。現役期インフレを 2% 適用した場合、2055年時点の生活費は 432 × 1.02^29 ≒ 770 万円になるはずで、**差額 338 万円/年**。現役期30年の合計影響額は数千万円規模。

### 03-I08（教育費インフレ未適用）
- **観察**: 2033-2043 の `leCost` は中学突入時 193、高校 191、大学 222 と推移するが、これは「学校段階による差」であって「時間経過によるインフレ」ではない。2039年の大学単価が 2026年ベースの名目単価のまま。
- **判定**: ✅ I08 がシナリオBでも発生。大学4年間の累積では 5～10% 程度の過小評価（13年後なのでインフレ 2% で約 25%）。

### 05-C03（住宅ローン残高線形近似）
- **観察**: 完済年 2056年に `leMortgage=140` が残存計上されている。**線形近似で `loanYears 35` を割っているため、2022開始 + 35年 = 2056完済だが、その年の返済もそのまま計上される**。
- **判定**: ✅ C03 の影響が定年年に重なる（2056年の年間支出を 140 万円過大評価）。退職初年の`annualExpense` 466 のうち 140 がローン返済だが、実際には 2055年末で完済しているはず。

### 06-C01（配偶者退職シミュでの pension_p 開始年不整合）
- **観察**: 退職シム 2056年（妻64歳）時点で `pension_p=120` がすでに計上開始。実際の `pensionAge_p=65` に基づくと妻の年金開始は 2058年であるべき。
- **判定**: ❌ 明確におかしい。2056-2057 の2年で妻年金 120×2=240 万円過大計上。

### 07-C01（投資プール枯渇後の幻影）
- **観察**: シナリオB では `depleted=false` が全年。発生していない。
- **判定**: ✅ このシナリオでは該当なし（資産潤沢）。

### 09-C01（現役期 vs 退職後のインフレ非対称）
- **観察**: 現役期 `annualExpense` 30年 432 固定 → 退職初年に **466 にジャンプ**。以降は退職インフレ 1% で年+1 ずつ増加（2057: 277 は `monthlyExpense 24 → 次年降下` で誤読注意、実は `baseExpense 217, recurringCost 60, leMortgage 0` に分解される）。
- **判定**: ✅ 現役期→退職期の境界で生活費の基準単価が段差的に変わっており、**30年の名目固定の反動で定年直後は実質生活水準が大幅ダウンに見える**。

### その他（シナリオB 固有）
1. **partnerWorkIncome 154 の無期限継続**: 退職シム全年（age 65～90）で計上され続ける。`cfe_sb_01.endAge=null` を尊重した実装だが、**FP 実務では現実的でない**（妻が本人90歳時で 88歳まで働く前提になる）。
2. **baseExpense 216 の内訳**: 退職期の月間生活費は `monthlyExpense=24` で年288 のはずが、baseExpense 216 と出ている。これは `expenseDecayRate 0.5` 等の減衰、または recurringCost などとの切り分け方が影響している可能性があり要確認。

## 主要な要改修点（優先度順）
1. **I01**: 現役期インフレ未適用 → 最大影響、全シナリオ共通
2. **06-C01**: 配偶者年金開始年のずれ → シナリオBで 240 万円相当（小規模だが明確なバグ）
3. **05-C03**: 完済年のローン返済残存 → シナリオB 定年年で 140 万円過大
4. **partnerWorkIncome 継続**: データ入力側の責任だが、UI で注意喚起が必要
5. **I08**: 教育費インフレ → 小規模だが教育費が高いシナリオ（D等）ではより顕在化
